重建Windows引导的一次操作

2019-04-04 Clover, UEFI, Windows, 引导 0 评论

由于电脑Windows和Linux系统都不在一个硬盘上，所以每个硬盘都有一个EFI分区。本着有一个引导分区就够了的原则就把装了Windows那块盘的ESP分区给删了，打算直接使用装Linux这块盘上的EFI分区来装所有系统的引导文件，遂重建Win引导。

Action

准备：

  • 装有PE环境的U盘一只
  • 计算机一台

启动PE系统
